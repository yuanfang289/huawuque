## public 目录简介
### 目录结构
```
- component
- fonts
- images
- plugins
- scripts
	- base
- styles
	- core        // 核心公用样式
	- control     // 表单控件样式
	- mixins      // 公用sass混合
	- plugins     // js 插件所需样式
```

### component
> 包括：对话框、上传组件、表单组件 等

### font
> 字体文件

### images
> 图片文件

### plugins
> 包括一些第三方的js插件

### scripts
> 包括一些js方法
>
> base文件夹包括：jquery、jquery-cookie、lazyload 等

### styles
> 公用样式
